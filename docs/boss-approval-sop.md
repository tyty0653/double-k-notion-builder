# Boss Approval SOP

Open the Pending Boss Approval view. Confirm customer, site, Quote ID, quote type, scope, quantities, prices, internal cost where relevant, discount amount, discount reason, requested by, and final amount.

- Approve: set Approval Status = Approved, Approved By, and Status = Boss Approved.
- Reject: set Rejected and record clear notes.
- Revision: set Revision Needed and return it to staff.

Review Project quotations and Variation Orders particularly carefully. Final quotations are not sent before boss approval in v1.
