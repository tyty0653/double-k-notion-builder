# Retail Quote SOP

1. Find or create the Customer and prevent duplicates by checking phone and company name.
2. Find or create the Site / Location.
3. Create a Retail quotation and manually enter the next official Double K Quote ID using the existing quotation numbering process.
4. Add Quote Items from Services & Price List, then enter quantity, price, discount amount, discount reason, and requester.
5. Set Status to Need Boss Approval and Approval Status to Pending Boss Approval.
6. After boss approval, set Boss Approved, send the final quote, and set Sent.
7. Follow up and record Accepted, Rejected, Expired, or Cancelled.
8. For Accepted, create a Retail Job and track appointment, Technician / Staff, photos, completion, payment, and next follow-up.

Staff never send a final quote before boss approval in v1.
