# User Role Guide

## Boss

Review pending quotes and variation orders, approve/reject/request revision, and monitor jobs, projects, payments, and import exceptions.

## Manager

Optionally check scope, pricing, costing, staff work, project progress, and data quality before boss approval.

## Admin / Office Staff

Maintain Customers and Sites, manually enter the official Quote ID, add quote items and discounts, submit for boss approval, send only approved quotes, create jobs/projects, track payments, and clean Excel Import Staging.

After setup, manually assign Boss, Manager, Admin/Office staff, PIC, Prepared By, Approved By, and Technician / Staff people fields.
