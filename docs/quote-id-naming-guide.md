# Quote ID Naming Guide

Double K already has an existing quotation numbering format. Continue that existing quotation numbering process.

- Quote ID is manually editable and searchable.
- Staff manually enter the next official company number.
- The builder does not generate, infer, increment, normalize for display, replace, or renumber it.
- Quote Year and Sequence No. are optional helpers only.
- For Excel imports, preserve Original Quote Number exactly in staging and Original Quote ID in the final quote.
- Copy it into Quote ID only after manual confirmation that it is official.
- Missing and duplicate IDs require manual review.

Sample values beginning `SAMPLE-` are fictional training labels, not a recommended company format.
