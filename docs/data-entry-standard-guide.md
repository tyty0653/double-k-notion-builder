# Data Entry Standard Guide

- Search phone, customer name, company, address, and Quote ID before creating a record.
- Use consistent customer and site names; keep full address in Address.
- Enter phone values as text when using Excel so leading zeroes survive.
- Use the existing quotation numbering process and type Quote ID manually.
- Record discount amount, discount reason, and requester together.
- Keep statuses current; do not treat notes as a substitute for workflow fields.
- Keep people fields empty until the actual workspace user is assigned.
- Treat Missing Quote ID and Duplicate Quote ID as manual review blockers.
- Never merge, delete, or overwrite an import row automatically.
