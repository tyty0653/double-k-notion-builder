# Manual Dashboard Setup Guide

The API creates dashboard pages, headings, links, and a few best-effort simple views. Build embedded linked database views in the Notion UI.

## Add a linked database view

1. Open a dashboard, type `/linked`, and select the required database.
2. Create a new view rather than changing the database's default view.
3. Add the section's status, type, date, or payment filter.
4. Sort urgent and recent work first. Group boards by Status or Project Stage.
5. Use table for operational lists, board for stages, calendar for appointments, and gallery only for useful photo records.

Boss approval views filter quotations to Status = Need Boss Approval and variation orders to the same status. Retail views filter Quote Type = Retail. Project views filter Quote Type = Project. Import review views filter Missing Quote ID and Duplicate Quote ID.

Check every relation and the Quote Items formula. Clean Excel Import Staging manually. Pilot with a small staff group, confirm permissions, retain original Excel backups, and then roll out safely.
