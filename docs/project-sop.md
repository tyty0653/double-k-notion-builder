# Project Quote SOP

1. Find or create Customer and Site records, then create the Project at Enquiry stage.
2. Record the site visit and collect BOQ, costing, drawings, PO, DO, and other Project Documents.
3. Build the Project quotation and manually enter the official Double K Quote ID.
4. Add Quote Items or complete project costing. Record all discounts and reasons.
5. Submit for boss approval. Project quotes always require boss approval.
6. After approval, send the final quote and update the quotation status.
7. If accepted, set Project Stage to Confirmed and track dates, documents, progress claims, payments, issues, and completion.
8. Create every scope change as a Variation Order; every VO requires boss approval before sending.
